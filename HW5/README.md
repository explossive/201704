# Задание
Написать свой тестовый фреймворк. Поддержать аннотации **@Test**, **@Before**, **@After**.
Запускать вызовом статического метода с
- (1) массивом классов с тестами,
- (2) именем package в котором надо найти и запустить тесты.

##### Ассерты:
- Assert.fail()
- Assert.true(...)
- Assert.notNull(...)
